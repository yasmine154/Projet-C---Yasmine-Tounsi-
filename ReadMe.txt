E-Bus YASMINE TOUNSI II1c
******************************
Cette application fait  : 
*Créer un nouveau voyageur / se connecter
*Créer un nouvelle réservation
*Afficher les détails d'un bus
*Afficher les détails d'une réservation
*Modifier une réservation 
*Faire gagner un voyage gratuit par coup de chance

*Lire le contenu des trois fichiers : "travelers.txt" , "reservation_data.txt" , "travelers.txt" et stocker les informations dans 3 tableaux de types convenables. 
A la fin de l'excecution, le programme fait enregistrer les informations stockées dans ces tableaux dans ces fichiers avec le mode "w".
